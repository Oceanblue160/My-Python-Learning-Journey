# 2. 项目介绍
## 2.1 主题
本项目代码是第一次课堂作业自己做出的题目（见附录）的答案，其中a.py是A题的答案，b.py是B题的答案，而Calculate_h-index.py是可以直接根据输入的论文引用情况输出h因子[^h因子]的计算器。
[^h因子]: h-index是科研实力的象征，其计算规则如下：若发布了n篇被引用n次以上的论文，则h因子的值等于n。
## 2.2 如何运行
所上传文件皆为py文件[^py文件]。
对于a.py直接运行可以输出结果，而b.py需要输入论文数量n，返回结果是斐波那契引用数的h因子。对于Calculate_h-index.py，可以直接输入一整行的论文引用数信息，会直接返回h因子。
[^py文件]:python源代码文件，可以使用python直接运行。
## 2.3 代码说明
三段代码的主体其实都是使用贪心算法计算h因子，即：
```python
n.sort(reverse=True)                #sort函数将n里面的元素从大到小排序  
for i in range(m):                  #循环m次（i从0开始，所以第一篇文章也是第0+1篇）  
    if n[i] < i + 1:                #如果第(i+1)篇文章的引用次数小于(i+1)，则后面所有文章引用次数都小于(i+1)  
        h = i                       #所以结果就是前面i篇引用次数大于i-1的文章。其实是简单的贪心算法。  
        break
```
其中b.py元素是从小到大排序，所以稍有区别，可以自行琢磨。
